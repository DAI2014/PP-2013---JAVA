Release Notes

Release Date: 31 December 2008
This is the 0.9.5 development release

Project: SwingX

This release is focused on bugfixes and increased quality of the code with minimal changes to the APIs.

JXLoginPane

Bug fixes for issues with resizing, transparency and flickering while changing views. Updated Javadoc for JXLoginDialog.

JXLabel, JXHeader

Fixed various visual issues mainly in UIDelegates and improved text layout functionality.

JXDatePicker, JXMonthView

Various fixes have been implemented for the above mentioned components to solve visual and behavioral issues reported by users.

JXComboBox, Autocomplete

Various fixes for usability issues issues reported by users.

JXTable, JXTreeTable, Highligters

Fixed Highlighter issues in JXTreeTable. Updated Javadoc. Solved issues with cell editing.

Project structure

Bean info have been split into separate jar to reduce total size of binary and to allow NetBeans and other UI to provide customized bean implementation to improve SwingX usage experience in the IDE visual editors.
